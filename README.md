## This is the `ecommerence ( mini shop )` example 

#### Prerequisites:

- Laravel >= 8
- PHP >= 7.3
- MySQL >= 5.6
- Composer
- NPM (Node.js)

#### - Set up the database and set appropriate variables in **.env** file (see **.env.example**)
```
php artisan key:generate
```


#### - remove livewire 

```
php artisan livewire:delete ( filename )
```
